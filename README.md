# Random-Password-Generator

To implement a website that generates Random Password based on the user's requirements.  

Front-end Technologies Used: HTML, CSS, JS  
No Backend and Databases are used.  
Deployment: This project has been deployed on Netlify.
Deployment Link: https://random-password-generator-a1b4.netlify.app/  

Project Timeline: Jun 11, 2026

Features Implemented:
1. Users can generate a random password based on the length that they need (between 8 and 32 - both inclusive)
2. Users can select the combinations of Upper Case characters, Lower Case characters, Digits and Special CHaracters according to their choice.
3. There is a button so that the users can access the GitHub repository of the project's source code.

Versions:  
*Ver 1.0.0*: This is the basic version of the project. This contains all the features mentioned above.
